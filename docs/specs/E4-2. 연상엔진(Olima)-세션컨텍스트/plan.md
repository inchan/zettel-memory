# 세션컨텍스트 기능 개발 계획

## 🎯 목표
연상엔진(Olima) / 세션컨텍스트: 세션 문맥/주제 기반 연상 검색과 자동 추천/리플렉션
- 로컬에 설치된 Olima 런타임(`@memory-mcp/assoc-engine`)과 연동하는 컨텍스트 저장소 유지

## 🔄 병렬 작업 구조 (Tracks)
### Track A: API/MCP
1. 입력/출력 스키마 정의 (Zod)
2. MCP 툴 구현 (stdin/stdout)
3. 에러/타임아웃 처리

### Track B: Core Logic
1. 핵심 로직 구현
2. 경계조건/리트라이
3. 로깅/메트릭 훅

### Track C: Storage/Index
1. 데이터 스키마/파일 구조
2. 인덱스 연동(필요 시)
3. 테스트 데이터 시드

### Track D: QA/Docs
1. 유닛/통합 테스트
2. 부하/회귀 테스트
3. 문서화/샘플 추가

## 📅 일정(예시, 병렬)
- 주1: 스키마/골격 코드, 테스트 스캐폴딩
- 주2: Core 구현 + 인덱스 연동
- 주3: 안정화/성능/문서
